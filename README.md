# 🐍 贪吃蛇游戏 (Snake Game)

一个用 Python 实现的经典贪吃蛇游戏，支持终端运行。

## 运行方式

```bash
python snake.py
```

## 操作说明

- **方向键**：控制蛇的移动方向
- **空格键**：暂停/继续游戏
- **Q 键**：退出游戏

## 特性

- 经典贪吃蛇玩法
- 计分系统
- 碰撞检测（墙壁和自身）
- 暂停功能

## 依赖

- Python 3.x
- curses（Linux/Mac 内置，Windows 需安装：`pip install windows-curses`）
